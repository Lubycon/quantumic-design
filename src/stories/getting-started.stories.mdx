# Getting Started

## **Installation**

Quantumic Design은 [npm 레지스트리](https://www.npmjs.com/package/@lubycon/ui-kit)를 통해 제공되고 있습니다. `npm` 또는 `yarn` 을 사용해서 패키지를 설치해주세요.

```bash
$ npm install @lubycon/ui-kit

// or

$ yarn add @lubycon/ui-kit
```

### Peer Dependencies

- `node >= 10.15`
- `react >= 17.0.1`
- `react-dom >= 16.9.0`
- (TypeScript 사용 시) `typescript >= 4.0.0`

## Design resources

Quantumic Design은 피그마의 컴포넌트를 기반으로 만들어졌으며, 해당 링크에서 피그마 컴포넌트를 다운로드 받을 수 있습니다

피그마 컴포넌트는 아래와 같은 상황에서 활용 가능합니다.

- Quantumic Design을 바탕으로 수정한 컴포넌트 정의가 필요한 경우
- 기획자 혹은 개발자 혼자 제품의 디자인적인 느낌을 고려하여 와이어프레임을 작성할 때
- 그 외 여러분이 필요한 모든 경우 😄
