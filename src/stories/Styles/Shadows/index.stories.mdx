# Shadow

You can use `Shadow` component when you want to express z-axis in 2D screen.

Humans recognize the height of objects according to the shape and outline of the shadow. If shadow is more clearly or smaller, you feel that object is in a low height.

`Shadow` component provides a 5 level shadow style according to this principle.

We recommend to use these predefined shadow levels to create a consistent height hierarchy in all parts of the application.
