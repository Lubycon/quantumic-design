# Input Field

- 사용자에게 텍스트 입력 및 수정을 가능케 하는 컴포넌트입니다
- 아이콘을 추가해 사용자의 의도에 부합하는 정보를 제공할 수 있습니다
- Default/Label/Placeholder/Disabled/Error/Types/Icon을 제공합니다

<br />

## Quick Start

Input 컴포넌트의 가장 기본 형태는 아래의 예시처럼 간단하게 사용해볼 수 있습니다.

```tsx
import React, { useState } from 'react';
import { Text, Input } from '@lubycon/ui-kit';

function Preview() {
  const [state, setState] = useState('');
  return (
    <div>
      <Text style={{ marginBottom: 16, display: 'block' }}>입력된 값: {state}</Text>
      <Input value={state} onChange={(e) => setState(e.target.value)} />
    </div>
  );
}
```
