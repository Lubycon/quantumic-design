# Text

`Text` component is a component that can be easily used anywhere the you need to enter text.

## Quick Start

You can use the `Text` component without any additional properties.

```tsx
import React from 'react';
import { Text } from 'quantumic-design';

function Preview() {
  return <Text>내용을 입력해주세요.</Text>;
}
```

Properties of `Text` component help you avoid declaring a font-related boilerplate CSS styles multiple times like below.

<Canvas>
  <Story name="Preview">
    <Text display="inline">display: Inline</Text>
    <Text size={24}>font-size: 24px</Text>
    <Text color="#48cfad">color: #48cfad</Text>
  </Story>
</Canvas>

As you can see, Text component doesn't have any CSS styles associated with the font. Therefore, It will inherit and use CSS styles declared at the root of your application.

### As

`Text` component will be rendered as p elements by default. but if you want to render it as other elements, you can use `as` props.
