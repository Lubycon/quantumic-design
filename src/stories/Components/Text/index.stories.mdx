# Text

Text 컴포넌트는 사용자가 텍스트를 입력해야 하는 곳이라면 어디든지 편리하게 사용할 수 있는 컴포넌트입니다.

## Quick Start

Text 컴포넌트는 별도의 속성 설정 없이 간단하게 사용해볼 수 있습니다. <br />
별도의 설정이 없는 경우
`typography`는 `p1`, `fontWeight`는 `regular`가 기본적으로 적용되며
`span` 엘리먼트로 렌더됩니다.

```tsx
import React from 'react';
import { Text } from 'quantumic-design';

function Preview() {
  return <Text>내용을 입력해주세요.</Text>;
}
```

## Font Weight

`fontWeight`는 `regular`가 기본입니다. 사용자는 4가지 속성을 의도에 따라 적용할 수 있습니다.

<Canvas>
  <Story name="Font Weight">

    </Text>
    <Text as="p">폰트 굵기 보통</Text>
    <Text as="p" weight={600}>
      폰트 굵기 굵게
    </Text>
    <Text as="p" weight={800}>

## As

Text 컴포넌트는 기본적으로 `span` 태그로 렌더되지만,
사용자의 의도에 따라 다양한 엘리먼트로 렌더하는게 가능합니다.

<Canvas>
  <Story name="As">
    <Text as="a" href="https://github.com" target="blank">
      앵커 태그입니다
    </Text>
    <Text as="h1">
      h1 태그로 렌더해도 typography 속성을 별도 지정해주지 않으면 typography는 p1이 적용됩니다
    </Text>
    <Text as="button">버튼으로 렌더하는 것도 가능하기는 합니다</Text>
  </Story>
</Canvas>

## Block

각 엘리먼트는 고유의 속성을 그대로 상속받습니다. block 레벨 엘리먼트는 `block` 속성을 가지고 있습니다.

<Canvas>
  <Story name="Block">
    <Text as="div">이 텍스트는 div태그로 렌더되며, display: block이 적용됩니다</Text>
    <Text as="p">이 텍스트는 p태그로 렌더되며, display: block이 적용됩니다</Text>
    <Text>이 텍스트는 span태그로 렌더되며, display: inline이 적용됩니다</Text>
  </Story>
</Canvas>
