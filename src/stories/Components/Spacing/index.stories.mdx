# Spacing

Spacing 컴포넌트는 요소와 요소 사이에 간격을 만듭니다.

특정 요소와 요소 사이의 간격이 명확하게 특정 컴포넌트 관심사가 아니라고 판단되는 경우, 각 컴포넌트에서 간격에 대한 관심사를 Spacing 컴포넌트에게 위임하세요.
