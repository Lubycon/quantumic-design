# Checkbox

기본적인 목적에 충실한 체크박스 컴포넌트를 제공합니다. 하나의 아이템이나 여러 개의 아이템을 선택하는 상황을 표현하는데 사용할 수 있습니다.

## Preview

간단한 라벨 정의만으로 체크박스 컴포넌트를 사용할 수 있습니다.

<Canvas>
  <Story name="Preview">
    <Checkbox label={'체크박스1'} />
  </Story>
</Canvas>
