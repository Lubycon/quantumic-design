# ProgressBar

ProgressBar 컴포넌트는 진행 상태를 시각적으로 나타내는 컴포넌트 입니다. Lubycon UI Kit에서는 현재 긴 막대 모양의 디자인만 제공하고 있습니다.

## Preview

가장 기본적인 형태의 ProgressBar 입니다. 별도의 텍스트 없이 ProgressBar만 렌더링 됩니다.

## Label

Label을 함께 노출합니다. 기본으로 지정된 위치는 ProgressBar의 상단이며, 값은 현재 진행 정도를 나타내는 숫자입니다. `top`, `bottom`, `left`, `right` 값을 통해 변경할 수 있습니다.
